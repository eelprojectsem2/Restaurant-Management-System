# Restaurant-Management-System

# Overview
The Restaurant Management System is a UI/UX design project developed in Figma that focuses on streamlining restaurant operations through a multi-role application. The system is designed for five key users—Customer, Waiter, Cook, Accountant, and Owner/Manager—each with a dedicated interface tailored to their specific tasks.


# Features 
* Multi-Role Access – Separate interfaces for Customer, Waiter, Cook, Accountant, and Manager
* Table Reservation – Easy booking with date, time, and availability selection
* Order Management – Real-time tracking and status updates of orders
* Kitchen Dashboard – Monitor order progress and inventory status
* Billing & Reports – View transactions and generate financial summaries
* Analytics Dashboard – Insights into performance, revenue, and operations
* Real-Time Updates – Live data visibility across all roles
* User-Friendly UI – Clean, intuitive, and role-based design system
